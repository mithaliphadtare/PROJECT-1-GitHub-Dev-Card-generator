🌟 About The Project

GitHub Dev Card Generator is a mini web application that creates visually appealing developer cards using GitHub profile information.

Simply enter a GitHub username and get an elegant profile card displaying key details such as repositories, followers, following, bio, and profile information.

---

✨ Features

🔍 Search GitHub users instantly

👤 Generate developer profile cards

📊 Display repositories, followers & following

🎨 Clean and modern UI

⚡ Fast and responsive design

🐳 Docker support

📱 Mobile-friendly interface
